what is :wqueue?

organize and find webpages.

:wqueue is a web app to save, organize and find webpages you want to access later. you can save webpages to your personal queue, adding helpful tags so you can remember what the page is all about. then you can search through your pages by content, titles, and tags.

how does it work?

type ':w'. on any web page, enter a few tags and hit enter to save that page. to exit the saving process, type 'q'. just like that, you've added a page to your queue.

why ":wqueue?".

:wqueue uses vim key bindings for it's core functionality. saving and closing are performed by typing ":wq". through our app users save webpages to their own queue, hence the name :wqueue.
